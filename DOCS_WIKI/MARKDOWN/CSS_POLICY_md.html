BEM Naming Convention

All components follow BEM (Block\_\_Element--Modifier):


Block:       .card, .sidebar, .top-bar, .feed-item
Element:     .top-bar__toggle, .top-bar__brand, .card__subtitle
Modifier:    .badge--active ŌåÆ .badge-active (flat variant)

Rules:

  • Double underscore `__` separates block from element
  • Double hyphen `--` for modifiers (flattened to single `-` for readability)
  • No nested element selectors in CSS (`.card h3` OK, `.card__title` preferred for new components)

    2. Low specificity: Class-only selectors, no ID styling

    3. Transitions everywhere: 0.12ŌĆō0.15s ease on all interactive elements

    4. Glass effects: `backdrop-filter: blur(8px)` on stats/info bars

    5. Hover micro-interactions: Cards lift (`translateY(-1px)`), feed items slide (`translateX(2px)`)

    6. Accessibility: `aria-label` on nav landmarks, `aria-expanded` on toggle button

    7. Top bar offset: `body {{ padding-top: 40px; }}` always matches `.top-bar` height
